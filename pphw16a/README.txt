This is the intruction of problem 6 of homework 1 of the course ID1217 Concurrent Programming.

This parallel multithreaded program finds all the palindromic words in file "words" 
and write the palindromic words to a results file "result.txt"

This program is compiled and only need to run directlly.

To build and run the project
The best choice to build and run project using eclipse 
Alternative: go into the directory of this project
command line : javac pWord.java
run : java pWord (argument 2 indicate how many threads will be used)

the second command argument controlls how many threads will be used( max 10 thread)
if run without the second argument, or a argument bigger than 10, the project will run in default 10 threads.
